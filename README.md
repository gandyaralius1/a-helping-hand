# A Helping Hand — Website

This repository contains the static website for A Helping Hand Re-Entry Program.

How to publish
- The site is static (HTML/CSS) and is ready to deploy to Netlify.
- Ensure the site root file is `index.html` (already provided).

Deploy on Netlify (quick)
1. Log in to Netlify → New site → Import from Git → connect GitHub → choose this repo.
2. Build command: leave blank. Publish directory: `/` (root).
3. Click "Deploy site". After deploy, add your custom domain in Site settings and enable HTTPS.

Netlify Forms
- The contact form is named `contact`. Netlify will capture submissions automatically after deployment.

Local preview
- Open `index.html` in your browser to preview locally.

Customize
- Edit `index.html` to update content, contacts, or branding colors.
